# Enlaces a los Datasets Utilizados

Para la creación de los requisitos de esta ontología hemos partido del dataset asignado y lo hemos extendido con datos adicionales relacionados para enriquecer el dominio:

1. **Dataset Principal (Asignado al Grupo 4)**
   * **Nombre**: DigiDB - Digimon Database
   * **Descripción**: Contiene las estadísticas base, tipos, atributos y etapas evolutivas de 341 Digimon.
   * **URL**: [https://www.kaggle.com/datasets/rtatman/digidb](https://www.kaggle.com/datasets/rtatman/digidb)

2. **Dataset Adicional (Extensión del dominio)**
   * **Nombre**: Digimon Support Movelist (Habilidades de support)
   * **Descripción**: Dataset complementario (incluido en la misma fuente principal) que lista las habilidades de support, con su descripción de lo que hace correspondiente.
   * **URL**: [https://www.kaggle.com/datasets/rtatman/digidb?select=DigiDB_supportlist.csv](https://www.kaggle.com/datasets/rtatman/digidb?select=DigiDB_supportlist.csv)

3. **Dataset Adicional (Extension del dominio)**
   * **Nombre**: Digimon Movelist (Habilidades de ataque). Este dataset sólo tiene los ataques que hacen daño al contricante y por lo tanto será fusionado con el de supportlist para tratar todos los ataques.
   * **Descripción**: Dataset complementario lista las habilidades que dañan al oponente con su descripción, tipo, coste, poder, atributo, heredable.
   * **URL**_ [https://www.kaggle.com/datasets/rtatman/digidb?select=DigiDB_movelist.csv](https://www.kaggle.com/datasets/rtatman/digidb?select=DigiDB_movelist.csv)
