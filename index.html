Resumen (Abstract)

DigiOnto es una ontología semántica diseñada para modelar el ecosistema de criaturas digitales (Digimon), sus líneas evolutivas y mecánicas de combate. Este recurso integra estadísticas base, atributos elementales y tipos para permitir aplicaciones como asistentes de construcción de equipos competitivos (Team Builder) y enciclopedias semánticas de combate.

Descripción

Esta ontología formaliza los conceptos extraídos de DigiDB, definiendo clases principales como ns:Digimon, ns:EvolutionStage, ns:Move y ns:Element. Además, establece axiomas lógicos para los atributos de combate (HP, Ataque, Velocidad) y modela el triángulo de ventajas de tipos (Vaccine, Virus, Data).

Para consultar el modelo visual sobre el que se fundamenta esta versión, puedes revisar el Diagrama Conceptual en Chowlk y el conjunto de reglas de validación en SHACL.
