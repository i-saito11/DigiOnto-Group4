# Casos de Uso: Ontología DigiOnto

Para guiar el desarrollo de esta ontología y justificar la integración de nuestras fuentes de datos, hemos definido los siguientes casos de uso:

## 1. Asistente de construcción de equipos competitivos (Team Builder)
Una aplicación dirigida a jugadores de videojuegos de Digimon que permite consultar qué criatura es óptima para un combate basándose en sus estadísticas. La ontología debe enlazar las estadísticas base de un Digimon (HP, Ataque, Defensa, Velocidad) con su Tipo y Atributo para calcular ventajas tácticas.

## 2. Enciclopedia Semántica de Combate
Un portal web donde los usuarios pueden buscar un Digimon y descubrir qué habilidades de soporte (Support Moves) puede aprender. El alcance de los ataques se ha acotado intencionadamente a las habilidades de soporte (curación y mejoras) para relacionar la clase principal de la criatura con habilidades utilitarias, filtrándolas por su coste de SP y el efecto que causan.
